## Governance Framework

### Veto Triggers (Absolute — cannot be overridden)
- Constitutional violation of any severity
- Regulatory exposure of any kind
- Risk score >7 without mitigation plan
- Spend that reduces runway below configured floor

### Review Mandatory (Board MUST review)
- Any single spend over configured threshold
- Any strategic pivot (new market, new product, major partnership)
- Any agent suspension or reinstatement
- Any constitutional amendment proposal
- Weekly: full fiduciary audit

### Review Optional (Board MAY review if flagged)
- Spend under threshold but in high-risk category (capital, experimental)
- Agent performance concerns escalated by department heads
- Pattern of small spends aggregating over threshold
